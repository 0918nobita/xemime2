# Xemime

implemented in Scala

## Run

```bash
sbt run
```

## Test

```bash
sbt test
```

## Generate executable JAR

Using ``sbt-assembly`` plugin

```bash
sbt assembly
```

Location: ``target/scala-2.13/xemime.jar``

Now we can run the executable:

```bash
java -jar target/scala-2.13/xemime.jar
```
